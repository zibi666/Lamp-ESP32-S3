# ESP32-S3 硬件引脚接线表

## 1. 毫米波雷达模块 (UART)

| 雷达模块引脚 | ESP32-S3 引脚 | 说明 |
|-------------|--------------|------|
| TX | GPIO5 | 雷达发送 → ESP32接收 |
| RX | GPIO4 | ESP32发送 → 雷达接收 |
| VCC | 3.3V / 5V | 根据雷达模块要求供电 |
| GND | GND | 共地 |

**配置参数**：
- 串口号：UART1
- 波特率：115200 (默认)
- 数据位：8
- 停止位：1
- 校验位：无

---

## 2. LED 灯带 (PWM 双色温)

| 灯带通道 | ESP32-S3 引脚 | LEDC 通道 | 说明 |
|---------|--------------|-----------|------|
| 黄光 (Warm) | GPIO20 | CHANNEL_0 | 暖色温控制 |
| 白光 (Cool) | GPIO19 | CHANNEL_1 | 冷色温控制 |
| VCC | 5V / 12V | - | 根据灯带规格供电 |
| GND | GND | - | 共地 |

**PWM 配置**：
- 分辨率：13-bit (0-8191)
- 频率：5000 Hz
- 定时器：TIMER_0
- 模式：低速模式

## ⚠️ 注意事项

1. **PSRAM 冲突**：ESP32-S3 使用 Octal PSRAM 时，GPIO35/36/37 被占用，不可用于其他功能
2. **电平匹配**：雷达模块需确认逻辑电平是否为 3.3V，否则需电平转换
3. **LED 灯带供电**：大功率灯带需独立电源，确保 GND 与 ESP32 共地
